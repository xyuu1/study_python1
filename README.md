# 数据分析技术学习平台

为商务数据分析与应用专业学生打造的现代化在线学习平台。

## 课程内容

### 课程模块
1. **Python基础** - 变量、数据类型、流程控制、函数
2. **数据来源和类型** - 常见数据源、数据格式、数据存储
3. **数据采集** - 文件读取、网络采集、API接口
4. **数据清洗** - 缺失值处理、数据转换、数据去重
5. **数据可视化** - Matplotlib基础、图表类型、数据展示

## 功能特点

- 🎨 赛博朋克科技风格设计
- 💻 交互式代码练习
- 📝 20道测验题（10选择+10判断）
- 📖 详细的答案解析
- ✅ 学习进度追踪
- 🌐 纯前端实现

## 技术栈

- React 18 + TypeScript
- Vite
- Tailwind CSS
- React Router DOM
- Lucide React

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build
```

## 项目结构

```
src/
├── components/     # 可复用组件
├── pages/         # 页面组件
├── data/          # 课程数据
├── hooks/         # 自定义Hooks
└── styles/       # 全局样式
```

## 学习路径

1. 访问首页了解课程概览
2. 选择章节开始学习
3. 完成代码练习
4. 参加章节测验
5. 查看答案解析
6. 追踪学习进度
