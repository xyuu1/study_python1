# Python 数据分析训练平台

专为商务数据分析专业大学生设计的 Python 数据分析技能训练网站。

## ✨ 功能特点

### 🎯 训练项目
- **销售数据分析** - Pandas + Matplotlib 数据处理与可视化
- **客户分群分析** - K-Means 聚类实现客户画像
- **财务报表分析** - 财务比率计算与趋势分析
- **市场调研数据分析** - 描述统计与交叉分析
- **库存优化预测** - 时间序列分析与预测模型
- **营销效果分析** - A/B 测试与归因分析
- **客户流失预测** - XGBoost 分类模型
- **价格优化分析** - 回归分析与需求预测
- **供应链数据分析** - 网络分析与路径优化
- **综合商业分析报告** - 多维度数据整合

### 📚 课程中心
- Python 基础入门
- Pandas 数据处理
- 数据可视化
- 统计分析
- 机器学习入门

### 👥 社区交流
- 热门讨论区
- 问题求助
- 资源共享

## 🛠️ 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite 6
- **样式框架**: Tailwind CSS 3
- **图标库**: Lucide React
- **状态管理**: Zustand
- **路由**: React Router DOM

## 🚀 快速开始

### 安装依赖
```bash
pnpm install
```

### 开发模式
```bash
pnpm run dev
```

### 构建生产版本
```bash
pnpm run build
```

### 预览生产版本
```bash
pnpm run preview
```

## 📁 项目结构

```
src/
├── components/          # UI 组件
│   ├── Header.tsx       # 头部导航
│   ├── Hero.tsx         # 首页横幅
│   ├── Stats.tsx        # 数据统计
│   ├── Projects.tsx     # 训练项目展示
│   ├── Courses.tsx      # 课程中心

│   └── Footer.tsx       # 页脚
├── data/
│   └── projects.ts      # 项目数据
├── hooks/               # 自定义 Hooks

├── pages/               # 页面组件
├── App.tsx              # 主应用组件
├── main.tsx             # 入口文件
└── index.css            # 全局样式
```
